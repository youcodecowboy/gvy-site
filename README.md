# Groovy Docs

A Next.js documentation application built with TypeScript, Tailwind CSS, and the App Router.

## How to Run

### Development

```bash
bun install
bun dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

### Build

```bash
bun run build
```

### Start Production Server

```bash
bun run start
```

### Lint

```bash
bun run lint
```
